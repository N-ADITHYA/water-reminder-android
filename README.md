A simple native Android app that helps users stay hydrated by sending periodic water reminders and tracking daily water intake.

---

##  Features

- Custom reminder intervals
- Background notifications (WorkManager)
- Live countdown timer
- Start / Stop reminder
- Daily water counter
- Automatic daily reset
- Persistent data storage
- Clickable notifications

---

##  Tech Stack

- Kotlin
- Jetpack Compose
- WorkManager
- Notifications API
- SharedPreferences

---

##  Download

Install the latest APK from the **Releases** section:

`app-release.apk`

> Enable *Install from Unknown Sources* on your device before installing.

##  Purpose

Built as a personal project to learn modern Android development using Kotlin and Jetpack Compose.
